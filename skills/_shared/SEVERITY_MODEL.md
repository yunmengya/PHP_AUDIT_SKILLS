# 严重等级模型（独立等级 + 组合等级）

## 三维评估
- 可达性：是否未认证可达、是否需要特殊权限。
- 影响范围：RCE/全库读取 > 敏感数据泄露 > 局部信息。
- 利用复杂度：单请求/低门槛 > 多步骤/高门槛。

## 决策树（简化版）
1. 未认证可达？
- YES: RCE/全库读取 → Critical，否则 → High
- NO: RCE/全库读取 → High，否则 → Medium/Low

## 编号体系
- C (9.0-10.0): 系统沦陷
- H (7.0-8.9): 重大损害
- M (4.0-6.9): 一定损害
- L (0.1-3.9): 低风险

## 独立等级 vs 组合等级
- 独立等级：假设不存在其他漏洞时的评级。
- 组合等级：考虑攻击链后的实际影响。
- 修复建议：优先修复能放大链路的“关键节点”。
